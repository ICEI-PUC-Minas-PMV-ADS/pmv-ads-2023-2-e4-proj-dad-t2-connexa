# CONNEXA

`Ánalise e desenvolvimento de Sistemas` 

`Projeto: Desenvolvimento de uma Aplicação Distribuída`

`Eixo 4`

O projeto Connexa é uma aplicação móvel e web desenvolvida para facilitar a coordenação e o compartilhamento de listas de compras entre grupos, seja em ambiente familiar, entre amigos ou colegas de trabalho. Connexa se destaca pela sua capacidade de unir os membros do grupo de forma eficiente, simplificando o processo de compras e melhorando a comunicação.
